# 用户操作手册

## 1. 系统概述

统计系统是一个数据统计与福利管理平台，用于记录和管理各分部人员的每周业务数据（收光、麦序、全麦），并根据可配置的奖励规则自动计算排名与福利。

### 核心功能

- **数据看板**：可视化展示本周数据汇总、周对比、Top3 排名
- **数据录入**：支持手动录入、Excel 导入、表格粘贴导入
- **排名与福利**：按分部分组排名，自动计算基础福利和排名奖励
- **人员管理**：管理人员名单及分部归属
- **系统设置**：配置奖励规则、管理账户、管理分部
- **数据导出**：导出 Excel / CSV 格式
- **系统通知**：查看排名公布、规则变更等通知

## 3. 登录与退出

### 3.1 登录

1. 打开系统网址，进入登录页面
2. 输入用户名和密码
3. 点击「登录」按钮

> 默认会长账户：用户名 `admin`，密码 `admin123`（首次部署后请及时修改）

### 3.2 退出

点击左侧导航栏底部的「退出登录」按钮即可退出系统。

## 4. 数据看板

数据看板是系统首页，展示当前选中周的统计数据概览。

### 4.1 切换周次

- 点击「‹」按钮查看上一周
- 点击「›」按钮查看下一周（不能超过本周）
- 点击「本周」快速回到当前周

### 4.2 KPI 卡片

展示四项关键指标：
- **分部人员数**：当前分部（或全部）的人员总数
- **本周总收光**：所有人员收光数量之和
- **本周总麦序**：所有人员麦序数量之和
- **本周总福利**：所有人员总福利之和

### 4.3 图表

- **本周数据汇总（按人员）**：柱状图展示每位人员的收光、麦序、全麦数据
- **周对比（本周 vs 上周）**：柱状图对比本周与上周的总量数据

### 4.4 Top3 排名

按分部分组展示每周前三名人员，包含排名、姓名、麦序和总福利。

### 4.5 分部筛选（仅会长）

会长可通过右上角下拉框切换查看不同分部，或选择「全部分部」查看汇总数据。

## 5. 数据录入

### 5.1 手动录入

1. 进入「数据录入」页面
2. 选择周次（默认本周）
3. 选择分部（仅会长需要选择，其他角色自动绑定）
4. 在录入表单中：
   - 选择人员（已录入本周数据的人员不会出现在列表中）
   - 输入收光、麦序、全麦数值（非负整数）
5. 点击「添加」按钮提交

### 5.2 编辑数据

1. 在数据表格中找到目标记录
2. 点击该行「编辑」按钮（铅笔图标）
3. 表单将填充该记录数据，修改后点击「保存」
4. 点击「取消」可放弃修改

### 5.3 删除数据

1. 在数据表格中找到目标记录
2. 点击该行「删除」按钮（垃圾桶图标）
3. 确认删除操作

> 仅会长和超管可删除数据，管理角色无删除权限。

### 5.4 Excel 导入

1. 点击「导入」按钮，打开导入弹窗
2. 选择「Excel上传」标签
3. 选择 `.xlsx` 或 `.xls` 文件
4. 点击「开始导入」

**Excel 格式要求：**
- 第一列：姓名
- 第二列：收光
- 第三列：麦序
- 第四列：全麦
- 第一行为表头，将被自动跳过

### 5.5 表格粘贴导入

1. 点击「导入」按钮，打开导入弹窗
2. 选择「表格粘贴」标签
3. 从 Excel 或其他表格软件复制数据，粘贴到文本框
4. 点击「开始导入」

**格式要求：**
- 支持 Tab 分隔或逗号分隔
- 第一行若包含「姓名」将被视为表头跳过
- 示例格式：`姓名\t收光\t麦序\t全麦`

### 5.6 数据导出

- **导出 Excel**：点击「导出Excel」按钮，下载 `.xlsx` 文件
- **导出 CSV**：点击「CSV」按钮，下载 `.csv` 文件

导出文件包含当前周次、选中分部的所有数据记录及福利计算结果。

## 6. 排名与福利

### 6.1 查看排名

1. 进入「排名与福利」页面
2. 选择周次（下拉框展示有数据的周次）
3. 查看排名表格，包含：排名、人员、收光、麦序、全麦、基础福利、排名奖励、总福利

### 6.2 福利计算规则

排名按分部分组，分部内按麦序降序排列：

- **基础福利** = 收光 × 收光比例 + 全麦 × 全麦比例
- **排名奖励**：
  - 第1名：获得第1名奖励
  - 第2名：获得第2名奖励
  - 第3名：获得第3名奖励
  - 第4名及以后：麦序达到达标阈值即获得达标奖励
- **总福利** = 基础福利 + 排名奖励

### 6.3 福利计算说明

页面底部展示当前分部的奖励规则详情，包括各项转换比例和奖励金额。

## 7. 人员管理

### 7.1 查看人员名单

进入「人员管理」页面，查看当前分部（或全部）的人员列表，包含序号、姓名、所属分部、本周数据状态。

### 7.2 添加人员

1. 点击「添加人员」按钮
2. 输入人员姓名
3. 选择所属分部（仅会长需要选择）
4. 点击「添加」完成创建

> 仅会长和超管可添加人员。

### 7.3 移除人员

1. 在人员列表中找到目标人员
2. 点击「移除」按钮
3. 确认移除操作

> 移除人员仅解除其与分部的关联，不会删除已有数据记录。

## 8. 系统设置

### 8.1 奖励规则设置

1. 进入「系统设置」页面
2. 在「奖励规则设置」区域：
   - 会长需先选择分部
   - 修改各项参数：
     - 收光转换比例
     - 全麦转换比例
     - 第1/2/3名奖励金额
     - 麦序达标阈值
     - 麦序达标奖励金额
3. 点击「保存规则」

> 仅会长和超管可修改奖励规则。

### 8.2 账户管理

1. 在「账户管理」区域：
   - 查看账户列表（用户名、角色、状态）
   - 点击「添加账户」创建新账户
2. 账户操作：
   - **启用/禁用**：点击状态按钮切换账户状态（禁用后无法登录）
   - **删除**：点击删除按钮移除账户

> 会长可管理所有超管和管理账户；超管仅可管理本分部的管理账户。

### 8.3 分部管理（仅会长）

1. 在「分部管理」区域：
   - 查看分部列表（名称、人员数、数据数）
   - 点击「添加分部」创建新分部
2. 删除分部：点击删除按钮（存在数据记录的分部无法删除）

### 8.4 通知列表

查看系统通知，包含通知内容、时间和已读状态。点击通知可标记为已读。

## 9. 通知查看

### 9.1 顶部通知栏

页面顶部右侧有通知铃铛图标，显示未读通知数量。鼠标悬停可查看通知列表（最多10条），点击通知可标记为已读。

### 9.2 设置页通知列表

在「系统设置」页面底部可查看完整通知列表（最多20条），点击通知标记为已读。

### 9.3 通知类型

| 类型 | 说明 |
|------|------|
| 排名公布 | 每周排名发布通知 |
| 规则变更 | 奖励规则修改通知 |
| 数据变更 | 数据录入/修改通知 |

## 10. 移动端使用

系统支持移动端访问，自适应不同屏幕尺寸：

- **手机（375px+）**：侧边栏默认隐藏，点击左上角菜单按钮展开
- **平板（768px+）**：侧边栏默认隐藏，点击菜单按钮展开
- **桌面（1024px+）**：侧边栏常驻显示

### 移动端操作提示

- 点击左上角「☰」按钮展开侧边栏导航
- 展开后点击导航项可跳转并自动收起侧边栏
- 点击侧边栏外的遮罩区域可关闭侧边栏
- 数据表格在窄屏下可横向滚动查看完整内容

## 11. 常见问题

### Q1: 忘记密码怎么办？

请联系会长或超管重置账户。会长账户密码遗忘需直接操作数据库或参考部署指南重新初始化。

### Q2: 登录提示「账户已被禁用」？

账户被管理员禁用，请联系会长或超管在「系统设置 → 账户管理」中启用账户。

### Q3: 录入数据时提示「请选择分部」？

会长角色需要先在页面顶部选择目标分部才能录入数据。其他角色若出现此提示，说明账户未关联分部，请联系会长配置。

### Q4: Excel 导入失败怎么办？

请检查 Excel 格式是否符合要求：
- 第一列为姓名，第二列收光，第三列麦序，第四列全麦
- 数值必须为非负整数
- 人员姓名需与系统中已有人员匹配

导入完成后系统会提示成功和失败的条数，失败详情可在浏览器控制台查看。

### Q5: 排名数据不正确？

排名按麦序降序排列，请确认：
1. 数据已正确录入（检查收光、麦序、全麦数值）
2. 奖励规则配置正确（检查「排名与福利」页面底部的规则说明）
3. 选择了正确的周次

### Q6: 如何查看历史数据？

在「数据录入」或「数据看板」页面，通过周次切换按钮（‹ / ›）查看不同周次的数据。「排名与福利」页面的周次下拉框会列出所有有数据的周次。

### Q7: 一个人员可以属于多个分部吗？

可以。系统支持人员与分部的多对多关系，同一人员可关联多个分部，在不同分部中分别录入数据。

### Q8: 导出的 Excel/CSV 包含哪些内容？

导出文件包含当前周次、选中分部的所有数据记录，含人员姓名、分部、收光、麦序、全麦、福利及录入时间等字段。
